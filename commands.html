slash commands

All commands are invoked with the / prefix inside Discord. Here’s the full list:

utility

  • /ping – Show latency
  • /invite – DM you the OAuth2 invite link
  • /servercount – Number of servers the bot is in
  • /about – About the bot
  • /help – Paged help embed

fun

  • /coinflip – Flip a coin
  • /avatar [user] – Show someone’s avatar
  • /hug [user] – Send a virtual hug
  • /8ball <question> – Magic 8-ball answer
  • /weather <city> – Current weather
  • /cat – Random cat pic
  • /define <word> – Dictionary definition
  • /remind <minutes> <text> – DM reminder
  • /uptime – How long the bot has been online
  • /poll "question" option1 | option2 … – Reaction poll (owner-only)

timezones

  • /tz set <alias> – Save your timezone
  • /tz change <alias> – Change your timezone
  • /tz current – Show your timezone
  • /tz supported – List supported aliases

moderation

  • /kick <user> [reason] – Kick a user
  • /ban <user> [reason] – Ban a user
  • /timeout <user> <minutes> [reason] – Timeout a user
  • /purge <count> – Bulk delete messages
  • /slowmode <seconds> – Set channel slowmode
  • /lock – Lock channel
  • /unlock – Unlock channel
  • /warn <user> <reason> – Warn a user
  • /warnings <user> – List warnings
